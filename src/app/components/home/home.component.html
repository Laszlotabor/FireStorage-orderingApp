Welcome to manulStorage storage manager

The "real life" version of this application(with auth) was built as a personal project to help myself at work(managing stocks in hospitality) and demonstrate my proficiency in Angular, Firebase (Firestore), and general frontend/backend integration. It's designed to manage custom storage units — allowing users to add items, place orders, and track them in real time.

Technologies Used

  • Angular (Standalone Components) – Modern architecture with streamlined modules
  • Firebase (Firestore) – Real-time database for storing items and orders
  • RxJS & Observables – For efficient asynchronous data handling
  • NgModel & FormsModule – Clean two-way data binding for forms
  • Angular Router – Navigation between views
  • TypeScript Interfaces – Strongly typed data models

Features

  • Create and manage storage units
  • Add or remove items
  • Place and cancel orders
  • Real-time sync with Firestore
  • Clear separation of services and UI logic

Purpose

This project serves as a practical showcase of full-stack thinking within a frontend-heavy Angular app. It demonstrates:

  • Clean and scalable code structure
  • Working with external services like Firestore
  • UX-minded dynamic component behavior
  • Problem-solving and debugging in a real-world context
